# Deployment

## Vercel (Demo/Sharing)

```bash
vercel --prod
```

**Use case:** Quick demos and sharing with collaborators

## AWS Lightsail (Research)

```bash
make quick-research
```

**Use case:** Research deployments with HIPAA compliance
